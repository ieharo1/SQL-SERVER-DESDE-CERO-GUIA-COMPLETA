# 🎯 EXE-ZOOM-ATACK

Proyecto de automatización y herramientas para Zoom desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Herramientas de automatización para la plataforma de videoconferencias Zoom, incluyendo scripts y ejecutables para facilitar tareas recurrentes.

---

## ✨ Características

- Automatización de reuniones
- Unirse automáticamente a reuniones
- Gestión de participantes
- Programación de tareas

---

## 🛠️ Stack Tecnológico

- Python
- Selenium
- AutoIt
- Zoom API
