Creación de Bases de Datos

Aprende a crear y configurar bases de datos en SQL Server

← Volver al inicio

Introducción

Una base de datos en SQL Server es una colección organizada de datos que se almacenan y acceden electrónicamente desde un sistema informático. En esta lección aprenderás los fundamentos para crear y configurar bases de datos.

💡 Nota: SQL Server utiliza el lenguaje T-SQL (Transact-SQL) para la gestión de bases de datos.

Sintaxis Básica

La instrucción CREATE DATABASE es la forma más simple de crear una base de datos:

SQL 
-- Crear una base de datos simple
CREATE DATABASE MiBaseDeDatos;
GO

-- Usar la base de datos creada
USE MiBaseDeDatos;
GO

Creación con Opciones Avanzadas

Puedes especificar archivos de datos, logs y otras opciones:

SQL 
CREATE DATABASE VentasEmpresa
ON PRIMARY 
(
    NAME = Ventas_Data,
    FILENAME = 'C:\SQLData\VentasEmpresa.mdf',
    SIZE = 10MB,
    MAXSIZE = 100MB,
    FILEGROWTH = 5MB
)
LOG ON 
(
    NAME = Ventas_Log,
    FILENAME = 'C:\SQLData\VentasEmpresa.ldf',
    SIZE = 5MB,
    MAXSIZE = 50MB,
    FILEGROWTH = 1MB
);
GO

Verificar Bases de Datos Existentes

SQL 
-- Listar todas las bases de datos
SELECT name, database_id, create_date 
FROM sys.databases 
ORDER BY name;

-- Verificar si una base de datos existe
IF DB_ID('MiBaseDeDatos') IS NOT NULL
    PRINT 'La base de datos existe';
ELSE
    PRINT 'La base de datos no existe';

-- Crear solo si no existe
IF NOT EXISTS (SELECT 1 FROM sys.databases WHERE name = 'MiBaseDeDatos')
BEGIN
    CREATE DATABASE MiBaseDeDatos;
    PRINT 'Base de datos creada exitosamente';
END
ELSE
BEGIN
    PRINT 'La base de datos ya existe';
END
GO

Modificar una Base de Datos

SQL 
-- Cambiar nombre de la base de datos
ALTER DATABASE MiBaseDeDatos 
MODIFY NAME = NuevaBaseDeDatos;
GO

-- Agregar un archivo de datos adicional
ALTER DATABASE VentasEmpresa
ADD FILE 
(
    NAME = Ventas_Data2,
    FILENAME = 'C:\SQLData\VentasEmpresa2.ndf',
    SIZE = 10MB,
    MAXSIZE = 50MB,
    FILEGROWTH = 5MB
);
GO

-- Cambiar opciones de recuperación
ALTER DATABASE VentasEmpresa 
SET RECOVERY FULL;
GO

Eliminar una Base de Datos

⚠️ Advertencia: La instrucción DROP DATABASE elimina permanentemente la base de datos y todos sus datos. ¡Usa con precaución!
SQL 
-- Eliminar una base de datos
DROP DATABASE MiBaseDeDatos;
GO

-- Eliminar solo si existe (SQL Server 2016+)
DROP DATABASE IF EXISTS MiBaseDeDatos;
GO

Ejercicio Práctico

Crea una base de datos para un sistema de biblioteca:

SQL - Ejercicio 
-- 1. Crea una base de datos llamada 'Biblioteca'
-- 2. Verifica que fue creada
-- 3. Cambia a esa base de datos
-- 4. Lista todas las bases de datos del servidor

-- Solución:
CREATE DATABASE Biblioteca;
GO

SELECT name FROM sys.databases WHERE name = 'Biblioteca';
GO

USE Biblioteca;
GO

SELECT name FROM sys.databases ORDER BY name;
GO
